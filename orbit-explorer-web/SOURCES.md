# Fuentes y criterios

La aplicación conserva la estructura del proyecto Java original y corrige la unidad de presión a atmósferas para mantener consistencia con valores como Tierra ≈ 1 atm y Venus ≈ 92 atm.

Fuentes de contexto científico consultadas:

- NASA Venus Facts: https://science.nasa.gov/venus/venus-facts/
- NASA Europa: https://science.nasa.gov/jupiter/jupiter-moons/europa/
- NASA Titan Facts: https://science.nasa.gov/saturn/moons/titan/facts/
- NASA Enceladus: https://science.nasa.gov/saturn/moons/enceladus/
- NASA Ocean Worlds: https://science.nasa.gov/solar-system/ocean-worlds/
- NASA Planet Compare: https://solarsystem.nasa.gov/planet-compare/

## Nota importante

Los campos `resources`, `waterPotential`, `radiationRisk`, `missionCost`, `fuel` y `duration` son indicadores o valores de simulación educativa. No deben presentarse como mediciones científicas exactas. Los parámetros pueden reemplazarse por datos investigados y documentados por el equipo.
