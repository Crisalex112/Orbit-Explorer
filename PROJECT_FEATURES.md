# Funcionalidades integradas

## Conservadas del proyecto Java

- Base de datos inicial de cuerpos celestes.
- Registro de nuevos cuerpos.
- Validación de campos y nombres duplicados.
- Evaluación de habitabilidad/explorabilidad.
- Comparación con la Tierra.
- Ranking de habitabilidad.
- Ranking de recursos.
- Clasificación por prioridades.
- Reporte detallado de puntajes y decisiones.

## Funcionalidades añadidas

- Interfaz web espacial adaptable a computadoras, tabletas y teléfonos.
- Navegación por módulos sin recargar la página.
- Persistencia mediante `localStorage`.
- Búsqueda y filtros de la base de datos.
- Índice ampliado con seis criterios ponderados.
- Indicadores booleanos de condiciones verdaderas y falsas.
- Historial de evaluaciones solicitadas por el usuario.
- Exportación de datos e historial en JSON.
- Gráficas cartesianas SVG interactivas.
- Simulación de misiones con presupuesto, combustible y tiempo.
- Estadísticas generales y promedios.
- Restablecimiento de la base de datos inicial.
- Pruebas automáticas del modelo.
- Servidor local sin dependencias externas.
- Diseño con estrellas animadas, nebulosas, órbitas y cuerpos celestes.
